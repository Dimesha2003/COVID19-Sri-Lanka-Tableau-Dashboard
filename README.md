
# COVID-19 Sri Lanka Dashboard (Tableau)

## 📊 Project Overview
Interactive Tableau dashboard visualizing COVID-19 cases across 26 districts in Sri Lanka from April 2020 to May 2021.

## 📈 Features
- National trend line chart
- Top 10 districts bar chart  
- Top 5 districts over time (line chart)
- District vs Month heatmap
- Interactive dashboard layout

## 🛠️ Tools Used
- Tableau Public
- CSV Data Cleaning & Pivot

## 📁 Files
- `COVID19_Sri_Lanka_Dashboard.twbx` - Tableau packaged workbook
